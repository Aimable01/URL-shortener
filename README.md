# Description

This is a simple url shortener

### Tech stack

- Nodejs
- Expressjs
- Mongodb
- shortid
- ejs

## Usage

You can clone the repository

```bash
git clone https://github.com/Aimable01/URL-shortener.git
```

```bash
npm install
npm run dev
```
